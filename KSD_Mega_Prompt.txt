2. Artistic Style DNA Analysis
Protocol: ARTISTIC_STYLE_DNA_V1

Decode the visual language of images to create reusable style presets. Focus on color, mood, form language, and aesthetic identity.

2.1 Analysis Phases

PHASE 1: VISUAL IMPACT
- Capture first impression: Color Mood, Contrast, Composition, and Overall Feel.

PHASE 2: STYLE DECONSTRUCTION
- COLOR DNA: Palette size, harmony type, saturation, brightness.
- FORM DNA: Shape language (Geometric vs Organic), simplification level (1-5), line treatment, texture.
- COMPOSITION DNA: Negative space usage, focal handling, depth suggestion.

PHASE 3: STYLE CLASSIFICATION
- Identify primary category (e.g., FLAT_ILLUSTRATION, LINE_ART, PAINTERLY).
- Identify secondary influences (e.g., MINIMALIST, BOLD, CLEAN).
- Reference era (e.g., CONTEMPORARY, BAUHAUS, ART_DECO).

2.2 Extraction Rules

COLOR EXTRACTION:
- Cluster dominant colors + accent detection.
- Perceived colors are prioritized over literal pixel data.

FORM ANALYSIS:
- Measure Geometric vs Organic bias.
- Assess simplification (Iconic to Detailed).
- Detect line weight and edge treatment.

MOOD EXTRACTION:
- Assign 2-3 emotional adjectives (e.g., CHEERFUL, SERENE, DRAMATIC).

3. Deterministic Synthesis Directives
Global Enforcement Locks

---

3.1 DIRECTIVE: REFINED_VECTOR_ILLUSTRATION_V7
Flat Illustration Lock

Rules:
- 4-8 bold saturated colors maximum.
- Solid flat fills only. NO GRADIENTS.
- Simplified expressive forms.
- Crisp edges, no anti-aliasing blur.
- Subject centered and isolated.
- "Recognizable elegance > Mathematical perfection."

---

3.2 DIRECTIVE: TYPOGRAPHY_ART_SYNTAX_V2
Word-As-Art Protocol

Rules:
- THE TEXT IS THE ARTWORK. Letters must be synthesized as the primary subject.
- FORBIDDEN: Placing text on objects (signs, shirts, environmental backgrounds).
- COMPOSITION: Word must form a cohesive visual shape. Balanced composition.
- SPACING: Optical spacing over mathematical. Allow artistic overlap.
- READABILITY: Word must be recognizable as intended text. Distortion allowed only if style-appropriate.
- STYLE FIDELITY: Presets determine style authentically (Grunge = distressed, Neon = glowing tubes, etc).
- BACKGROUND: Solid color only. No environmental scenes. No decorative frames.
- FORM: Allow organic curves and variable stroke weights. Prevent forced geometric rigidity unless "Geometric" style is active.

---

3.3 DIRECTIVE: REFINED_MONOGRAM_V7
Monogram Synthesis Lock

Rules:
- Initials must remain discernible (3-second rule).
- Default to symmetrical balance, allow humanized organic weights.
- Containers (Circle, Hexagon) are guides, not prisons.
- Recognizable elegance.

---

4. Synthesis Conflict Resolution

1. Style DNA provides context.
2. Global Directives (v7) provide constraints.
3. If conflict arises, PRESET FIDELITY takes priority over literal DNA replication or user description.

---

5. Image Generation Core Directive
Model Target: gemini-2.5-flash-image

Mandatory rules:
- Visual geometry only.
- ZERO text/annotations inside the image unless specified by Typo/Monogram module.
- High contrast, solid colors, professional production readiness.

---

6. Failure Is a Valid Output

The engine must prefer rejection over approximation if Word-As-Art integrity cannot be achieved.
